# RF Signal Strength Prediction using MATLAB

## Project Overview

This project predicts RF signal strength based on distance from a transmitter using the Free Space Path Loss model. The simulation is implemented in MATLAB and visualizes how signal attenuation increases as the distance from the mobile tower increases.

## Objective

The objective of this project is to analyze and visualize RF signal attenuation using mathematical path loss models.

## Tools Used

* MATLAB

## Methodology

The project uses the Free Space Path Loss formula to calculate signal loss with respect to distance and frequency.

Steps involved:

1. Define the distance range from the transmitter.
2. Define operating frequencies (900 MHz and 1800 MHz).
3. Calculate path loss using the path loss formula.
4. Plot graphs to visualize signal attenuation.

## Output

The program generates a graph that shows:

* Distance from tower (km) on X-axis
* Path loss (dB) on Y-axis

The graph compares signal loss for:

* 900 MHz
* 1800 MHz

## Results

The simulation shows that as the distance from the transmitter increases, the RF signal strength decreases due to higher path loss.
